# От Populus Romanus к Populus Christianus: понятие «народ» у Фомы Аквинского в свете дистрибутивной семантики

Этот репозиторий содержит код и данные для воспроизведения результатов исследования, представленного в одноименной научной статье.

## Аннотация

Цель исследования – эмпирическая верификация и количественное измерение структурной трансформации семантического поля слов, связанных с понятием «народ», в творчестве Фомы Аквинского по сравнению с языковой нормой классической латыни. С помощью диахронических моделей векторных представлений слов (word embeddings) мы оцениваем масштаб семантического дрейфа, реконструируем топологию семантического пространства и даем содержательную интерпретацию выявленным сдвигам.

## Структура репозитория

-   `/scr`: Основной Python-скрипт `analysis.py` для проведения анализа.
-   `/notebooks`: Jupyter-ноутбук `latin_semantic_shift_analysis.ipynb` с исследовательским кодом.
-   `/data`: Папка для векторных моделей (скачиваются автоматически при первом запуске).
-   `/heatmaps`: Папка с полными версиями тепловых катр.
-   `/results`: Папка для сгенерированных графиков и таблиц.
-   `requirements.txt`: Список необходимых Python-библиотек.

## Воспроизведение результатов

Для полного воспроизведения анализа и генерации всех визуализаций, представленных в статье, выполните следующие шаги.

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/alexeyvkuznetsov/latin-historical-semantics.git
cd latin-historical-semantics
```

**2. Создайте виртуальное окружение и установите зависимости:**
```bash
python -m venv env
source env/bin/activate  # Для Windows: env\Scripts\activate
pip install -r requirements.txt
```

**3. Запустите анализ:**
```bash
python src/analysis.py
```

После выполнения скрипта произойдет следующее:
-   В папку `data/` будут автоматически скачаны две модели векторных представлений (`OperaLatina.vec.txt` и `OperaMaiora.vec.txt`).
-   В папке `results/` будут созданы все графики (`.png`) и таблицы (`.csv`, `.txt`), использованные в статье.
